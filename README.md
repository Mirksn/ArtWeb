# ManeVArt

A dynamic art gallery website showcasing artwork collections with admin capabilities for content management.

## Features

- User Authentication
- Admin Dashboard
- Dynamic Gallery Management
- Responsive Design
- Image Slideshow

## Technologies Used

- HTML5/CSS3
- JavaScript (ES6+)
- Firebase (Authentication & Storage)
- Webpack
- Responsive Design
